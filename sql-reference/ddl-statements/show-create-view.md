# SHOW CREATE VIEW {#show-create-view}

Shows the SQL statement that creates the specified view.

展示创建视图的语句。

## Synopsis\(文法\) {#synopsis}

```
SHOW CREATE VIEW view_name
```

### Parameters\(参数\) {#parameters}

`view_name 视图名称`

## Examples {#examples}

```
SHOW CREATE VIEW orders_by_date
```
