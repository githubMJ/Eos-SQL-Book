# SHOW COLUMNS {#show-columns}

Lists the columns in the schema for a base table or a view.

列出基表或视图的架构中的列。

## Synopsis\(文法\) {#synopsis}

```
SHOW COLUMNS IN table_name|view_name
```

### Parameters\(参数\) {#parameters}

`table_name 表名`

`view_name 视图名`

## Examples {#examples}

```
SHOW COLUMNS IN table01;
```
